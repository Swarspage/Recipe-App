## Packages
@dnd-kit/core | Core drag and drop functionality for the weekly planner
@dnd-kit/sortable | Sortable list behaviors for the weekly planner
@dnd-kit/utilities | Utilities for drag and drop components
framer-motion | Page transitions and complex micro-interactions
recharts | Beautiful data visualization for recipe macro charts
lucide-react | Premium iconography 

## Notes
- App uses a custom auth context wrapper checking /api/auth/check on load
- Web Speech API is used for the pantry voice input (requires modern browser)
- Real-time match scoring happens on the client side for immediate UI feedback
